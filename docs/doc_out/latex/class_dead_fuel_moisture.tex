\doxysubsection{Detailed Description}
Determines moisture content of dead and down woody fuels from ambient air temperature, relative humidity, insolation, and rainfall. 

\doxylink{class_dead_fuel_moisture}{Dead\+Fuel\+Moisture} is an implementation of Nelson\textquotesingle{}s dead fuel moisture model \doxysectlink{citations_nelson2000}{Nelson, Ralph M. Jr. 2000.}{1}. This section describes \doxylink{class_dead_fuel_moisture}{Dead\+Fuel\+Moisture} class usage only; for a more complete discussion of the Nelson model, see \doxysectlink{_dead_fuel_moisture}{Dead Fuel Moisture}{0}.
